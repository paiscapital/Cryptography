# Pigpen Cipher

### What is?
> A geometric simple substitution cipher, which exchanges letters for symbols which are fragments of a grid.

![alt text](https://miro.medium.com/v2/resize:fit:1400/1*4tHgmNenIr6KT586F03CCg.jpeg)
